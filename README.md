# FluxM2M

Mainflux [LwM2M](https://omaspecworks.org/what-is-oma-specworks/iot/lightweight-m2m-lwm2m/) server.

### Installation

Installing FluxM2M is trivial [`go get`](https://golang.org/cmd/go/):
```bash
go get github.com/mainflux/fluxm2m
$GOBIN/fluxm2m
```
